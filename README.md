This repository is created for the purpose of sharing the objective EDR evaluation questionnaire. EDR stands for Endpoint Detection and Response. This evaluation is created based on my exepriences from the many EDR evaluations conducted and and several market research & resources available on the internet such as [Red Canary EDR Evaluation guide.](https://redcanary.com/resources/guides/edr-evaluation-guide/)
